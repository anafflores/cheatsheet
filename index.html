CSS Selectors and Specifity

Selectors

A selector is used to target the specific HTML elements to be styled. It does not include the angle brackets {}

Selector Description Example
Class name Class selectors select all elements with a matching class. .jumbo { text-size: 100px;}
Element selectors You are able to select HTML elements first by simply using the name of the element. h1 {color: blue;}
ID selectors ID selectors are used to select only a single item on a page. They will ONLY select the first element with a matching ID. #thatThingINeededToStyle {color: blue; font-size: 24px;}
Attribute selectors It can be used to target HTML elements that already contain attributes input[type="text"] { width 100px;}
Child selectors You can also use multiple selectors to get the exact elements you want, by using parental nesting. By using the “greater-than” symbol, you can select only the direct children of an element, going down only one level. ul a { text-underline: none;}
Universal selector The universal selector (*) may be used to select all the elements in a particular range. body * { color: red; }
(this example will select children of the body)
Pseudo class selectors Pseudo Selectors can be used to narrow down a selection with certain rules. li:first-child { color: red;}
(This selects only li elements that have no elements before them)
